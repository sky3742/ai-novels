# AI Novel Multi-Agent System

Language: Simplified Chinese

Agents:

- architect
- plot_manager
- writer
- editor

Workflow:

1 architect creates story setup
2 plot_manager initializes database
3 writer writes next chapter
4 editor improves chapter
5 plot_manager updates database

Rules:

- 所有小说内容必须使用中文
- 每章 1500-2200 字
- 节奏快
- 对话多
- 每章结尾必须有悬念

Files:

story setup:
memory/story_setup.md

story database:
memory/story_db.md

chapters:
chapters/chapter_xxx.md
