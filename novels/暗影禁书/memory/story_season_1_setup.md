# 第一卷：学院风云

## 卷目标
- 林影进入魔法学院，开始系统学习魔法
- 意外触发禁书，暗影体质觉醒
- 与艾莉丝相识，建立初步关系
- 引起暗影议会注意

## 主要事件
1. 入学测试：林影隐藏实力，以普通成绩入学
2. 图书馆奇遇：发现禁书，暗影能量爆发
3. 教会调查：艾莉丝奉命调查禁书事件
4. 第一次冲突：暗影议会试探性袭击
5. 身份暴露危机：林影暗影体质险些暴露

第1章：禁书觉醒 - 林影在图书馆触发禁书
第2章：光明圣女 - 艾莉丝前来调查

第6章：教会的怀疑 - 艾莉丝对林影产生怀疑
第7章：学院比赛 - 学院内部魔法比赛
第8章：禁书秘密 - 林影研究禁书能力
第9章：暗影议会袭击 - 第一次正式袭击
第10章：身份危机 - 暗影体质险些暴露
第11章：艾莉丝的信任 - 两人关系进展

第13章：禁书第二本 - 发现第二本禁书线索

第15章：暗影议会现身 - 与暗影议会正面冲突

第20章：卷终决战 - 学院保卫战，第一卷结束
